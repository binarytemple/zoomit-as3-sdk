Classpublic final class AsyncRequest
InheritanceAsyncRequest flash.events.EventDispatcher
Implements IDisposable

Language Version : ActionScript 3.0

The AsyncRequest class provides an abstraction of messaging for API call invocation. An AsyncRequest allows multiple requests to be made on a remote destination and will dispatch either a ResultEvent when the remote request is completed or a FaultEvent if the request fails.
